## 摘要
尽管多模态大语言模型（MLLMs）展现出卓越的能力，但在长上下文场景下仍面临“视觉褪色”（visual fading）的问题。具体而言，随着文本序列的增长，模型对视觉标记（visual tokens）的注意力会逐渐减弱，导致生成的文本脱离视觉约束。我们将这种退化归因于多模态旋转位置编码（Multimodal RoPE）固有的归纳偏置，即随着视觉标记与文本标记之间距离的增加，它会惩罚跨模态注意力。为解决这一问题，我们提出了跨模态距离不变位置编码（DIPE），这是一种简单而有效的机制，它根据模态交互对位置编码进行解耦。DIPE 保留了模态内交互的自然相对位置以维持局部结构，同时为跨模态交互强制执行一种锚定的感知邻近性。该策略有效地减轻了基于跨模态距离的惩罚，确保视觉信号无论上下文长度如何都能保持感知一致性。实验结果表明，通过将 DIPE 与多模态 RoPE 集成，模型在长上下文场景中保持了稳定的视觉定位（visual grounding），在显著缓解视觉褪色的同时，保留了在标准短上下文基准测试上的性能。代码可在 https://github.com/lchen1019/DIPE 获取。

## Abstract
Despite the remarkable capabilities of Multimodal Large Language Models (MLLMs), they still suffer from visual fading in long-context scenarios. Specifically, the attention to visual tokens diminishes as the text sequence lengthens, leading to text generation detached from visual constraints. We attribute this degradation to the inherent inductive bias of Multimodal RoPE, which penalizes inter-modal attention as the distance between visual and text tokens increases. To address this, we propose inter-modal Distance Invariant Position Encoding (DIPE), a simple but effective mechanism that disentangles position encoding based on modality interactions. DIPE retains the natural relative positioning for intra-modal interactions to preserve local structure, while enforcing an anchored perceptual proximity for inter-modal interactions. This strategy effectively mitigates the inter-modal distance-based penalty, ensuring that visual signals remain perceptually consistent regardless of the context length. Experimental results demonstrate that by integrating DIPE with Multimodal RoPE, the model maintains stable visual grounding in long-context scenarios, significantly alleviating visual fading while preserving performance on standard short-context benchmarks. Code is available at https://github.com/lchen1019/DIPE.

*   **研究背景**：现有的 MLLM 多采用多模态旋转位置编码（MRoPE）。MRoPE 继承了 RoPE 的**长程衰减（Long-term Decay）**特性，旨在通过距离惩罚来捕捉语言的局部性。然而，在多模态场景下，随着生成文本增多，文本与图像 token 之间的相对距离单调增加，这种位置编码机制会错误地将图像视为“遥远的过去”而加以屏蔽。
*   **核心动机**：人类在观察图像并进行长篇论述时，视觉参考是持续存在的，不会随时间推移而“退色”。作者旨在重新设计位置编码，消除这种不合理的跨模态距离惩罚。

### 2. 方法论：跨模态距离不变位置编码（DIPE）
*   **核心思想**：将位置编码根据**模态交互关系**进行解耦。对于模态内部交互，保留相对位置以维持结构；对于跨模态交互，强制执行恒定的“感知距离”。
*   **关键技术细节**：
    *   **顺序位置编码（SPE）**：应用于**模态内（Intra-modal）**注意力（如文本对文本、图像对图像）。它沿用标准的 MRoPE 索引分配，确保语言的局部性和图像的 2D 空间结构得到保留。
    *   **锚定位置编码（APE）**：应用于**跨模态（Inter-modal）**注意力（如文本对图像）。DIPE 为每个模态段分配一个固定的“锚点”索引（通常是该段起始 token 的位置）。当文本查询图像时，无论文本生成到多长，其感知的图像距离始终保持在锚点位置，从而消除距离惩罚。
    *   **算法流程与实现**：
        1.  **双视图查询生成**：在 Forward 过程中，为 Query 向量生成两个视图：一个应用 SPE，一个应用 APE。Key 向量统一应用 SPE。
        2.  **解耦注意力计算**：利用掩码（Mask）分别计算模态内和跨模态的注意力得分。
        3.  **高效融合（LogSumExp Trick）**：为了兼容 FlashAttention，作者利用其返回的 LogSumExp 统计量，通过 Sigmoid 加权融合两个内核的输出。这种方法**不增加 FLOPs**，且完全兼容 **KV Cache**。

*   **理论优雅且直观**：准确识别了 RoPE 长程衰减与多模态持久观察需求之间的结构性矛盾。

### 8. 不足与局限
*   **模型规模覆盖**：实验主要集中在 3B 及以下的小规模模型，尚未在大规模模型（如 72B 或更高）上验证其边际收益。
*   **注意力稀释问题**：虽然 DIPE 解决了位置编码导致的“强制惩罚”，但无法解决 Softmax 机制在长序列下天然的“注意力稀释”（Attention Dilution）问题，即随着 token 总数增加，分配给每个 token 的权重均值必然下降。
*   **极端长度挑战**：在 64K 以上的极端长度下，模型性能仍受限于语言模型本身的上下文窗口容量，DIPE 无法逾越 LLM 本身的长度极限。
